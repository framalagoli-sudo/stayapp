# StayApp

Piattaforma SaaS per strutture ricettive italiane. L'ospite inquadra un QR code e installa una PWA sul telefono.

## Stack tecnico

- **Frontend**: React (PWA)
- **Backend**: Node.js
- **Database**: Supabase
- **Pagamenti**: Stripe Connect
- **Hosting**: Vercel

## Piani

- Base
- Standard
- Premium
- Enterprise

## Livelli di accesso

| Ruolo | Descrizione |
|---|---|
| Super Admin | Francesco |
| Admin Gruppo | Gestisce più strutture |
| Admin Struttura | Gestisce una singola struttura |
| Staff | Operativo nella struttura |
| Ospite | Utente finale via PWA |

## MVP

- QR code per struttura
- App installabile (PWA)
- Guida struttura
- Pannello admin
- Richieste alla reception
- Upselling base

## Approccio di sviluppo

Sviluppo agile — funzionalità aggiunte iterativamente in base al feedback dei clienti.
